📖 功能说明

🔍

核心功能

  • 输入长链接，生成短链接
  • 支持自定义短码（如 my-link）
  • 访问短链接自动重定向到原始链接
  • 自动记录每次访问次数
🛡️

安全设计

  • 仅支持 http/https 协议，防止非法协议
  • 短码格式严格校验，防止注入
  • 输入长度限制，防止恶意超长输入
  • 所有操作记录日志，支持回溯
⚙️

技术栈

  • 后端：Python Flask
  • 数据库：SQLite (+ SQLAlchemy ORM)
  • 前端：原生 HTML/CSS/JavaScript
  • API 格式：RESTful JSON
